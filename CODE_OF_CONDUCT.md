# Code of Conduct

## 1. Purpose

This project is committed to providing a welcoming and inclusive environment for everyone.

---

## 2. Expected Behavior

- Be respectful and professional.
- Accept feedback gracefully.
- Use inclusive language.
- Focus on collaboration.

---

## 3. Unacceptable Behavior

- Harassment or discrimination.
- Offensive comments or personal attacks.
- Spam or disruptive behavior.
- Sharing private information without consent.

---

## 4. Enforcement

Maintainers may take appropriate action including:

- Warning
- Temporary restriction
- Permanent ban

---

## 5. Reporting

If you encounter misconduct, report it to:

your-email@example.com

All reports will be handled confidentially.

---

By participating in this project, you agree to follow this Code of Conduct.
